# Synaps-Hardware-Sales-Profit-Analysis-Dashboard
This project is an interactive Power BI Dashboard developed to analyze the sales and profitability of Synaps Hardware Store between 2014 and 2017. The dataset was provided by Easy Technologies Academy for learning and portfolio development.

The dashboard transforms raw sales data into meaningful business intelligence, enabling users to monitor key performance indicators (KPIs), evaluate regional and product performance, understand customer segments, and identify trends that support strategic decision-making.

Project Objectives

The objectives of this project were to:

Analyze overall sales and profitability.
Monitor monthly sales performance.
Compare sales across regions.
Evaluate customer segment performance.
Identify top-performing product categories.
Analyze profit contribution by sales teams.
Develop an interactive business intelligence dashboard using Power BI.
